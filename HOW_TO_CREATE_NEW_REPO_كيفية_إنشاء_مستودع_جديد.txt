الخطوة 1: إنشاء Repository جديد
────────────────────────────────
1. اذهب إلى: https://github.com/new
2. اسم Repository: اختر اسمًا (مثال: my-research-project)
3. الوصف: "Interactive Academic Research Website"
4. اختر: Public (عام)
5. ✅ لا تضف README أو .gitignore أو License (سنرفعها يدويًا)
6. انقر: "Create repository"

الخطوة 2: رفع الملفات
──────────────────────
1. في صفحة Repository الجديد، انقر: "uploading an existing file"
2. فك ضغط الملف: COMPLETE_REPOSITORY_v3.6_مستودع_كامل.tar.gz
3. افتح مجلد: NEW_REPOSITORY_COMPLETE
4. اسحب وأفلت جميع الملفات (28 ملف) إلى GitHub
5. اكتب رسالة Commit: "Initial commit - Complete research website v3.6"
6. انقر: "Commit changes"

الخطوة 3: تفعيل GitHub Pages
──────────────────────────────
1. اذهب إلى: Settings (في Repository)
2. انقر: Pages (من القائمة الجانبية)
3. Source: اختر "main" branch
4. Folder: اختر "/ (root)"
5. انقر: "Save"
6. انتظر 5 دقائق

الخطوة 4: زيارة الموقع
