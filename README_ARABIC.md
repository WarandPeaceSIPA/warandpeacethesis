## 🚀 البدء السريع

### الخيار 1: واجهة GitHub الويب (الأسهل - 5 دقائق)

1. اذهب إلى مستودع GitHub الخاص بك
2. انقر على "Add file" → "Upload files"
3. اسحب وأفلت جميع الملفات من هذا المجلد
4. اكتب رسالة الالتزام: "Initial commit - Research website v3.6"
5. انقر على "Commit changes"
6. انتظر 5 دقائق حتى يتم بناء GitHub Pages
7. قم بزيارة: `https://[your-username].github.io/[repository-name]/`

### الخيار 2: سطر أوامر Git

```bash
# تهيئة المستودع
git init

# إضافة جميع الملفات
git add .

# الالتزام
git commit -m "Initial commit - Research website v3.6"

# إضافة المستودع البعيد (استبدل برابط مستودعك)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# الدفع إلى GitHub
git push -u origin main
```

### الخيار 3: سكريبت النشر السريع (تلقائي)

```bash
# جعل السكريبت قابلاً للتنفيذ
chmod +x quick-deploy.sh

# تشغيل السكريبت
./quick-deploy.sh
```
