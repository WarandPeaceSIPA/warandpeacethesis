# 🚀 دليل البدء السريع | Quick Start Guide

## للنشر على GitHub Pages خلال 5 دقائق:

### الخطوة 1: إنشاء حساب GitHub
- اذهب إلى: **https://github.com**
- انقر "Sign up" وأكمل التسجيل

### الخطوة 2: إنشاء Repository
1. انقر زر **"+"** في الأعلى
2. اختر **"New repository"**
3. اسم الـ Repository: **`warandpeacethesis`**
4. اجعله **Public** ✅
5. انقر **"Create repository"**

### الخطوة 3: رفع الملفات
1. في صفحة الـ Repository، انقر **"uploading an existing file"**
2. اسحب جميع الملفات من مجلد `warandpeacethesis-github`:
   - ✅ index.html
   - ✅ script.js
   - ✅ styles.css
   - ✅ papers-database.json
   - ✅ package.json
   - ✅ README.md
   - ✅ .gitignore
   - ✅ CNAME
3. انقر **"Commit changes"**

### الخطوة 4: تفعيل GitHub Pages
1. اذهب إلى **Settings** (الإعدادات)
2. في القائمة الجانبية، انقر **"Pages"**
3. في **"Source"**:
   - اختر **"main"** branch
   - اختر **"/ (root)"**
4. انقر **"Save"**

### الخطوة 5: انتظر دقيقة واحدة ⏱️
- سيظهر رابط أخضر في الأعلى
- الرابط سيكون: **`https://YOUR_USERNAME.github.io/warandpeacethesis`**

## 🔄 لتحديث الموقع لاحقاً:

1. اذهب إلى الـ Repository
2. انقر على الملف الذي تريد تعديله
3. انقر على أيقونة القلم (Edit)
4. قم بالتعديلات
5. انقر "Commit changes"
6. التحديث سيظهر خلال دقيقة!
