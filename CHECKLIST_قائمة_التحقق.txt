   □ index.html موجود

┌─────────────────────────────────────────────────────────────────────────────┐
│ 🚀 CHECKLIST 2: أثناء النشر (During Deployment)                            │
└─────────────────────────────────────────────────────────────────────────────┘

إذا اخترت الطريقة 1 (GitHub Web):
□ تسجيل الدخول إلى GitHub
□ فتح Repository: https://github.com/WarandPeaceSIPA/warandpeacethesis
□ النقر على "Add file" → "Upload files"
□ سحب وإفلات جميع الملفات
□ كتابة رسالة Commit واضحة
□ النقر على "Commit changes"

إذا اخترت الطريقة 2 (Git Command Line):
□ فتح Terminal
□ الانتقال إلى مجلد المشروع
□ تنفيذ: git add .
□ تنفيذ: git commit -m "Update website v3.6"
□ تنفيذ: git push origin main

إذا اخترت الطريقة 3 (السكريبت):
□ فتح Terminal
□ الانتقال إلى مجلد warandpeacethesis-github
□ تنفيذ: bash quick-deploy.sh
□ إدخال اسم المستخدم
□ إدخال Personal Access Token

إذا اخترت الطريقة 4 (GitHub Desktop):
□ فتح GitHub Desktop
□ إضافة Repository المحلي
□ كتابة رسالة Commit
□ النقر على "Commit to main"
□ النقر على "Push origin"

│ ⏳ CHECKLIST 3: بعد النشر مباشرة (Post-Deployment - First 10 Minutes)     │
└─────────────────────────────────────────────────────────────────────────────┘

□ 1. الانتظار 5-10 دقائق
   (GitHub Pages يحتاج وقت لبناء الموقع)

□ 2. فتح الرابط في المتصفح:
   https://warandpeacesipa.github.io/warandpeacethesis/

□ 3. التحقق من تحميل الصفحة الرئيسية
   □ الصفحة تظهر بشكل صحيح
   □ الألوان والتصميم سليم
   □ الشعار يظهر (إن وجد)

□ 4. التحقق من التنقل
   □ زر "الرئيسية" يعمل
   □ زر "الفصول" يعمل
   □ زر "الاستشهادات" يعمل
   □ زر "المنهجية" يعمل
   □ زر "تواصل" يعمل
   □ زر "لوحة البيانات" يعمل

□ 5. التحقق من لوحة البيانات
   □ النقر على "لوحة البيانات"
   □ الصفحة تفتح بشكل صحيح
   □ الرسوم البيانية تظهر (6 رسوم)
   □ البيانات تتحمل بشكل صحيح

□ 6. فتح Console للتحقق (اضغط F12)
   □ لا توجد أخطاء حمراء (Errors)
   □ تظهر رسالة "📊 إحصائيات الزيارة"
   □ تظهر رسالة "🎯 تتبع الحدث"

┌─────────────────────────────────────────────────────────────────────────────┐
│ 📊 CHECKLIST 4: التحقق من الإحصائيات (Analytics Verification)             │
└─────────────────────────────────────────────────────────────────────────────┘

□ 1. Google Analytics 4
   □ فتح https://analytics.google.com
   □ اختيار Property الخاص بك
   □ الانتقال إلى "Realtime"
   □ يجب أن ترى زيارتك الحالية (1 visitor)
   □ التحقق من الصفحات المعروضة
   □ التحقق من الأحداث (Events)

□ 2. Microsoft Clarity
   □ فتح https://clarity.microsoft.com
   □ اختيار Project الخاص بك
   □ الانتقال إلى "Dashboard"
   □ يجب أن ترى "Recording" جديد
   □ النقر على Recording لمشاهدة الفيديو
   □ التحقق من Heatmaps (قد يحتاج 24 ساعة)

□ 3. النظام المحلي
   □ فتح Console (F12)
   □ يجب أن ترى: "📊 إحصائيات الزيارة"
   □ يجب أن ترى: "عدد الزيارات: X"
   □ عند النقر على أي زر، يجب أن ترى: "🎯 تتبع الحدث"
