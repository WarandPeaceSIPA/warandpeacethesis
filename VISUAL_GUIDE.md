## 📋 الخطوات المصورة

### الخطوة 1: إنشاء حساب GitHub

#### 1.1 صفحة التسجيل
```
🖼️ ستجد:
- حقل Email
- حقل Password  
- حقل Username
- زر "Sign up for GitHub"
```

**ما يجب فعله:**
- أدخل بريدك الإلكتروني: `zmalessi@dah.edu.sa`
- اختر كلمة مرور قوية
- اختر اسم مستخدم (مثل: `zaher-alessi`)

---

### الخطوة 2: إنشاء Repository

#### 2.1 صفحة Repository الجديد
```
🖼️ ستجد:
- حقل "Repository name"
- خيار Public/Private
- خيارات إضافية (README, .gitignore, License)
- زر أخضر "Create repository"
```

**ما يجب فعله:**
- اسم الـ Repository: `warandpeacethesis`
- اختر: ✅ Public
- لا تختر أي خيارات إضافية
- انقر "Create repository"

---

### الخطوة 3: رفع الملفات

#### 3.1 صفحة الرفع
```
🖼️ ستجد:
- منطقة كبيرة للسحب والإفلات
- رابط "choose your files"
- حقل "Commit changes"
- زر أخضر "Commit changes"
```

**ما يجب فعله:**
- اسحب جميع الملفات إلى منطقة الرفع
- أو انقر "choose your files" واختر الملفات
- انقر "Commit changes"

**الملفات المطلوبة:**
```

✅ README.md (3 KB)
✅ .gitignore (165 B)
✅ CNAME (28 B)
```

### الخطوة 4: تفعيل GitHub Pages

#### 4.1 صفحة Settings
```
🖼️ ستجد:
- قائمة جانبية طويلة
- خيار "Pages" في المنتصف
- أيقونة صفحة بجانبها
```

**ما يجب فعله:**
- انقر على "Settings" في الأعلى
- ابحث عن "Pages" في القائمة الجانبية
- انقر عليها

#### 4.2 صفحة Pages
```
🖼️ ستجد:
- قسم "Build and deployment"
- قائمة "Source" 
- قائمة "Branch"
- زر أزرق "Save"
```

**ما يجب فعله:**
- Source: اختر "Deploy from a branch"
- Branch: اختر "main"
- Folder: اختر "/ (root)"
- انقر "Save"

#### 4.3 رسالة النجاح
```
🖼️ ستظهر رسالة خضراء:
"✅ Your site is live at https://YOUR_USERNAME.github.io/warandpeacethesis"
```

**الرابط النهائي:**
- انسخ الرابط من الرسالة الخضراء
- أو استخدم: `https://YOUR_USERNAME.github.io/warandpeacethesis`

---

### الخطوة 5: زيارة الموقع

#### 5.1 الصفحة الرئيسية
```
🖼️ يجب أن ترى:
- شعار جامعة دار الحكمة في الأعلى
- عنوان الرسالة بخط كبير
- اسم الجامعة بالذهبي
- اسم الباحث والمشرف
- زرين للتنقل
```

#### 5.2 شريط التنقل
```
🖼️ يجب أن ترى:
- شعار الجامعة الصغير
- قائمة: الرئيسية، الملخص، الإطار النظري، قاعدة البيانات، الفصول، تواصل
- ألوان بنية وذهبية
```

#### 5.3 قاعدة البيانات
```
🖼️ يجب أن ترى:
- صندوق بحث
- فلاتر (السنة، الترتيب)
- قائمة بالأوراق البحثية
- أزرار الصفحات في الأسفل
```

## 💡 نصائح للتصوير

عند التقاط لقطات الشاشة:

### الجودة:
- استخدم دقة عالية (1920x1080 أو أعلى)
- تأكد من وضوح النص
- لا تستخدم Zoom كبير جداً

### المحتوى:
- اجعل الصورة تشمل العنصر الكامل
- تجنب قص الأجزاء المهمة
- أضف أسهم أو تعليقات إذا لزم الأمر

### التنظيم:
- رقّم الصور حسب الترتيب
- احفظها في مجلد منفصل
- استخدم أسماء واضحة (مثل: step1_signup.png)
